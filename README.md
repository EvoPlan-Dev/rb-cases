
# rb-cases v2 Case Opening System for QBCore

Give your players an exciting case-opening experience with smooth animations, rarity tiers.

Features:

Smooth CSS spinner animation with winner locked to center
5 rarity tiers with color-coded rewards: Common, Uncommon, Rare, Epic, Legendary
Supports both item and money (cash) rewards with random amount ranges (min/max)
Fully config-driven — add or edit cases and rewards in config.lua, no code changes needed
Item images automatically loaded from qb-inventory
Comes with 6 ready-to-use cases: Weapon Case, Pig Case, Simpel Case, Golden War Case, Tech Case & Funny Case

# Dependencies:
QBCore Framework
oxmysql
qb-inventory
